# Human Audit Workflow

This folder is reserved for generated or externally supplied human-audit artifacts.

The implemented scaffold supports:

- blinded artifact sampling,
- de-identified reviewer IDs,
- four-level ordinal scoring,
- pre-adjudication reviewer ratings,
- deterministic adjudication,
- raw agreement,
- quadratic weighted Cohen's kappa,
- outcome distributions.

Do not store identifiable reviewer information here. IRB/OHR status should be documented separately only after advisor or OHR guidance is available.
